Сверстай секцию header (используй семантический тег) внутрь ставь div с класоом container

Внутри блока с класоом container сделай верстку по следующему описанию:
    - вставь блок обертку
    - внктри блока обертки размести 2 блока:
        - в левый размести логотип (в теге img) обернутый в ссылку картирку возьми public\img\logo.svg
        - В правом размести блок с меню внутрь которого размести тег nav в котором расположи меню в виде ul списка в нем должны быть следующие пункты:Концепция, Прайс, О компании, Контакты (с ul списку не применяй стили он будет выводиться из админки)

При верстке используй БЭМ натацию

Стилизуй сверстанный блок начиная с блока с классом header__wrapper согласно следующему описанию:
    - блок header__wrapper flex; flrx-wrap:wrap; gap:20px;
    - логотип внутри header__left должен быть шириной 150 px
    - Блок header__right должен быть flex; align-item: end;
    - Блок header__right должен иметь нижнее подчеркивание в 1px сплошной линией цвет $main_color
    - блок header__nav должен быть отцентрован по горизонтали
